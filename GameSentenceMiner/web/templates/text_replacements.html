What are Text Replacements?

Text replacements help you automatically fix common errors and clean up text captured from games. This is especially useful for:

  • OCR Error Correction: Fix text recognition mistakes from screenshots
  • Text Standardization: Replace inconsistent character names or terms
  • Regex Pattern Support: Use "re:" prefix for advanced pattern matching (e.g., "re:.{3,}" for sequences)
  • Game-Specific Fixes: Handle unique formatting issues from different games

Pro Tip: Replacements are applied in the order they appear. Use regex patterns for complex transformations.
