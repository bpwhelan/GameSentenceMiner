OBS Capture (Required)

❓ Setup Capture adds Window/Game Capture for windows, or Video Capture Device for capture cards and cameras.
GSM prefers Game Capture when it works, and capture-card scenes use Video Capture Device.
On Linux/Steam Deck, Setup Capture currently uses XComposite Window Capture.
Capture-card scenes are created without automatic window-title scene switching.
Overlay support on Linux and macOS is experimental. You can still try Run Overlay. Click this warning for setup details.
